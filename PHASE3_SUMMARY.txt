================================================================================
                            API TESTING
================================================================================

GET /api/deals?page=1&limit=20&stage=NEGOTIATION&search=enterprise
- Returns: List of deals with pagination
- Status: 200 OK
- Includes: customer, createdBy, expectedClosureDate

POST /api/deals
- Creates: New deal with provided fields
- Returns: Created deal object
- Status: 201 Created
- Validates: name, customerId, value required

GET /api/deals/:id
- Returns: Full deal detail with customer, createdBy, activityLogs
- Status: 200 OK
- Soft delete check: Excludes deleted records

PATCH /api/deals/:id
- Updates: Any deal field (partial update)
- Returns: Updated deal object
- Status: 200 OK

DELETE /api/deals/:id
- Soft deletes: Sets deletedAt timestamp
- Returns: Success message
- Status: 200 OK

POST /api/deals/:id/move
- Moves: Deal to new SPANCO stage
- Returns: Updated deal object
- Status: 200 OK
- Validates: Stage must be one of 6 SPANCO stages
