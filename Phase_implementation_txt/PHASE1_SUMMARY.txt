🚀 HOW TO USE

1. START THE APP:
   npm run dev
   Open: http://localhost:3000

2. LOGIN:
   Email: admin@company.local
   Password: password

3. NAVIGATE TO LEADS:
   Click "Leads" in sidebar or go to: http://localhost:3000/leads

4. CREATE NEW LEAD:
   Click "+ New Lead" button
   Fill in: Name, Email, Company, Phone (optional), Source
   Click "Create Lead"

5. VIEW LEAD DETAIL:
   Click "View" in table or after creation
   See: Lead info, status, qualification notes, recent follow-ups

6. UPDATE LEAD:
   Click "Edit" in Status & Qualification section
   Change status and add BANT notes
   Click "Save Changes"

7. FILTER LEADS:
   Use dropdowns for Status and Source
   Type in search box for text search
   Click "Search"

8. DELETE LEAD:
   Click "Delete" in table
   Confirm deletion

🔍 DATABASE

Leads are stored in PostgreSQL with:
  • 15 fields (id, name, email, phone, company, source, status, score, etc.)
  • Relationships: assignedTo (user), linkedCustomer, followUps
  • Soft delete (deletedAt field)
  • Timestamps (createdAt, updatedAt)
  • Full-text search capable
