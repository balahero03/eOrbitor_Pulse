================================================================================
                            TECHNICAL STACK
================================================================================

BACKEND:
- API Route: Next.js 14 App Router
- Database: Prisma ORM with PostgreSQL
- Authentication: JWT token validation
- Aggregations: Prisma groupBy for distributions
- Date filtering: createdAt range queries
- Relationships: Multi-table joins (User, Deal, Lead, Order, Customer, Ticket, Task)

FRONTEND:
- Framework: React 18 with Next.js 14
- Language: TypeScript for type safety
- Styling: Tailwind CSS with gradient cards
- State Management: React useState/useEffect hooks
- Date Input: HTML5 date pickers
- Currency Formatting: Intl.NumberFormat (INR)
- Progress Bars: Pure CSS with width calculations
- Data Tables: HTML tables with hover states

DATABASE QUERIES:
- groupBy operations for aggregations
- aggregate functions (_count, _sum, _avg)
- whereClause with date ranges
- orderBy for sorting results
- take/skip for pagination
- Relationship includes for enriched data

CALCULATED METRICS:
- Win rate = (deals won / total deals) * 100
- Conversion rate = (converted / assigned) * 100
- Revenue share = (customer revenue / total revenue) * 100
- Average order value = total revenue / order count
- Pipeline concentration by stage
