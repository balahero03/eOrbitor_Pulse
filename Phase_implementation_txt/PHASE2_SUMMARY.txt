================================================================================
                            API TESTING
================================================================================

GET /api/customers?page=1&limit=20&industry=Technology&search=abc
- Returns: List of customers with pagination
- Status: 200 OK
- Includes: activeDealCount, contactCount calculated

POST /api/customers
- Creates: New customer with provided fields
- Returns: Created customer object
- Status: 201 Created
- Validates: companyName required

GET /api/customers/:id
- Returns: Full customer detail with contacts, deals, activity logs
- Status: 200 OK
- Soft delete check: Excludes deleted records

PATCH /api/customers/:id
- Updates: Any customer field (partial update)
- Returns: Updated customer object
- Status: 200 OK

DELETE /api/customers/:id
- Soft deletes: Sets deletedAt timestamp
- Returns: Success message
- Status: 200 OK

POST /api/customers/:id/contacts
- Creates: New contact for customer
- Returns: Created contact object
- Status: 201 Created
- Validates: firstName and email required

PATCH /api/customers/:id/contacts/:contactId
- Updates: Contact information
- Returns: Updated contact object
- Status: 200 OK

DELETE /api/customers/:id/contacts/:contactId
- Soft deletes: Sets deletedAt on contact
- Returns: Success message
- Status: 200 OK
