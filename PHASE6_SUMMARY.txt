================================================================================
                            API TESTING
================================================================================

GET /api/tasks?page=1&limit=20&status=TODO&priority=HIGH&assignedToId=USER_ID
- Returns: List of tasks with filtering
- Status: 200 OK
- Includes: title, description, status, priority, dueDate, assignedTo, relatedDeal

POST /api/tasks
- Creates: New task
- Returns: Created task object
- Status: 201 Created
- Validates: title, assignedToId required

GET /api/tasks/:id
- Returns: Full task detail
- Status: 200 OK
- Includes: all fields, assignedTo, relatedDeal, createdBy

PATCH /api/tasks/:id
- Updates: Task fields (status, priority, dueDate, description, tags, etc)
- Returns: Updated task
- Status: 200 OK

DELETE /api/tasks/:id
- Deletes: Task
- Returns: Success message
- Status: 200 OK

POST /api/tasks/:id/complete
- Marks: Task as completed
- Returns: Updated task with status=COMPLETED, completedAt timestamp
- Status: 200 OK

GET /api/followups?page=1&limit=50&type=CALL&fromDate=YYYY-MM-DD&toDate=YYYY-MM-DD
- Returns: List of follow-ups with filtering
- Status: 200 OK
- Includes: type, deal, lead, scheduledDate, actualDate, notes, outcome

POST /api/followups
- Creates: New follow-up
- Returns: Created follow-up object
- Status: 201 Created
- Validates: dealId, type, scheduledDate required

GET /api/followups/:id
- Returns: Full follow-up detail
- Status: 200 OK
- Includes: all fields, deal, lead, createdBy

PATCH /api/followups/:id
- Updates: Follow-up fields (actualDate, duration, notes, outcome, nextAction)
- Returns: Updated follow-up
- Status: 200 OK

DELETE /api/followups/:id
- Deletes: Follow-up
- Returns: Success message
- Status: 200 OK
