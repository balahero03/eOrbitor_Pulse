================================================================================
                            API TESTING
================================================================================

GET /api/orders?page=1&limit=20&status=PENDING&paymentStatus=PENDING
- Returns: List of orders with pagination
- Status: 200 OK
- Includes: customer, quotation, deal, totals, payment status

POST /api/orders
- Creates: New order from quotation or deal
- Returns: Created order object
- Status: 201 Created
- Validates: customerId, totalAmount required
- Auto-generates: orderNumber (ORD-YYYY-00001)

GET /api/orders/:id
- Returns: Full order detail
- Status: 200 OK
- Includes: customer, quotation, deal, timeline

PATCH /api/orders/:id
- Updates: Order fields (status, payment, dates, PO)
- Returns: Updated order
- Status: 200 OK

DELETE /api/orders/:id
- Deletes: Order
- Returns: Success message
- Status: 200 OK

POST /api/orders/:id/confirm
- Confirms: Order (PENDING → CONFIRMED)
- Returns: Updated order
- Status: 200 OK

POST /api/orders/:id/fulfill
- Fulfills: Order (CONFIRMED → FULFILLED)
- Returns: Updated order
- Status: 200 OK
- Records: deliveryDate

POST /api/orders/:id/payment
- Records: Payment
- Returns: Updated order
- Status: 200 OK
- Auto-updates: paymentStatus (PENDING/PARTIAL/COMPLETED)
